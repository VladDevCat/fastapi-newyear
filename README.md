Проект реализует простой веб-сервер на Python 3.11 с использованием фреймворка FastAPI.
Сервер обрабатывает HTTP GET-запрос по маршруту /info и возвращает JSON с количеством дней до наступления Нового года.

Пример ответа:

{
    "days_before_new_year": 308
}

Используемые технологии:

- Python 3.11;
- FastAPI;
- Uvicorn;
- Docker.

Запуск проекта:

- Сборка образа: docker build -t fastapi-newyear .
- Запуск контейнера: docker run -p 4200:4200 fastapi-newyear
- Проверка: curl http://localhost:4200/info

Проект контейнеризирован.
Dockerfile использует образ python:3.11-slim, устанавливает зависимости и запускает сервер через Uvicorn.
Это позволяет запускать приложение на любом устройстве, где установлен Docker, без установки Python.

Маршрут API
GET /info
Возвращает количество дней до наступления Нового года.
Пример запроса: curl http://localhost:4200/info

Контейнеризация обеспечивает:
- переносимость между устройствами
- одинаковую среду выполнения
- отсутствие необходимости устанавливать Python
- изоляцию приложения

Принцип работы:
Клиент отправляет HTTP GET-запрос.
Uvicorn принимает запрос.
FastAPI сопоставляет маршрут.
Выполняется функция обработки.
Возвращается JSON-ответ.
